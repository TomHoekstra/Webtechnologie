Beste beoordelaar,

De navigatie van deze site is gebouwd met behulp van angular routing. Dit zorgt ervoor dat wij een masterpage hebben die ervoor zorgt dat je de header en footer niet in meerdere pagina's hoeft te stoppen.
Het enige nadeel hiervan is dat de website gehost moet worden op een website of op een lokale webserver. Mocht dat niet lukken om dat zelf te doen dan is de website hier te bekijken: 
https://wt-practicum1.firebaseapp.com/#/home
De website is geschikt gemaakt voor alle devices. Dus schroom niet om hem te bekijken op je telefoon of tablet.

Het practicum is ingeleverd zonder de videobestanden. deze stond submit niet toe (> 20mb). Op mijn (Tom Hoekstra) GitHub is de repository als zip te downloaden
